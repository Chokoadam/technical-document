Introduction

C++ is a general-purpose programming language that was developed as an enhancement of the C language to include object-oriented paradigm. It is an imperative and a compiled language.

While containing majority of the features in C, it adds object-oriented-programming (OOP) by offering classes. In return, this provides four features commonly present in OOP: abstraction, encapsulation, inheritance, and polymorphism.

How it started? It was developed by Bjarne Stroustrup at Bell Labs in 1983!

Prerequisite Knowledge

The prerequisite knowledge you need to know:

  • Some programming experience. This could be with languages such as javascript, python, java, etc.
  • A general understanding of the C language and its syntax.
  • A general understanding of object-oriented-programming.

First Code

The keyword "std::" is used before io statements cin (input) and cout (output) as it indicates that these statements are in the namespace whose name is by default, std. There is a way to bypass doing this by declaring "using namespace std" in the header. However, this is not best practice as it can lead to naming collisions of namespaces.

Now to actually write some code. Here is the syntax and an example of how to write an output statement:

std::cout << "Hello World!";

Here is the syntax and an example of an input statement:

int input;
std::cout << "Enter an integer number:";
std::cin >> input;

Variables

Variables are a named container with data associated with it (strings, integers, chars, etc.).

That said, there is not an unified way to declare a variable as there are many data types in c++. However, below I will show an example of how to declare variables of some basic data types:

Integers:

int x = 10;

Doubles:

double y = 10.75;

Strings:

std::string name = "Adam";

Data Types

Data types are categories of data based on their attributes. They determine the values a variable can store and the operations that can be performed on them.

According for GeeksforGeeks, there are seven data types:

  • Integer: Integer numbers such as 1,2 and 3.
  • Characters. A single character such as 'a', 'b', 'c'. Can accomadate 256 values.
  • Boolean. True or false.
  • Floating Point. Decimal value numbers of 32 bits (4 bytes).
  • Double Floating Point. Can store twice the number of decimal values as floats, 64 bits (8 bytes).
  • Void. Has no value.
  • Wide Characters. Takes up more space than characters, but it can accept more values (65536 values).

Conditional Statements

A conditional statement tells a program to execute an action depending on whether a condition is true or false.

The conditional logic for c++ follows the same mathematical conditional logic. Here is a quick run down of conditional operators:

  • <: Less than.
  • >: Greater than.
  • <=: Less than or equal to.
  • >=: Greater than or equal to.
  • ==: Is equal to.

There are operators that can be used to combine conditional statements. "&&" means AND and "||" means OR.

Now, here is a list of various conditional operators and how to write them:

  • If Statements:
    • if(x < y) return 20;
  • Else-If Statements:
    • else if(x < z) return 30;
  • Switch Cases:
    • switch(day) {
case 1:
std::cout << "Monday";
break;
}
